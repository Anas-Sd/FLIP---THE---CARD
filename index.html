Game Rules

1. Basic Game Rules

You will be Flipping 2 cards each turn. If the symbols match, the pair stays revealed. If not they will be flipped back. Match all pairs to win.

2. How Lifes Work

You start with 10 lifes. Each time you flip two non-matching cards, you lose one life. For a correct match the lifes are gonna remain same. When lives reach 0 the game ends.

3. How Score Works

Each correct match increases your streak. The score gained equals the current streak value: first consecutive match +1, next consecutive +2, then +3, etc. A mismatch resets the streak so the next correct match starts at +1 again.
